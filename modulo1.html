Descripcion del Módulo

El Objetivo de este modulo es desarrollar la competencia de desarrollo de aplicaciones de software como medio para controlar, mecanizar o automatizar las tareas típicamente humanas, utilizando un lenguaje de programación orientado a objetos y un modelo de desarrollo.

Se abordaran temas extenos, se busca obtener un manejo intermedio de los lenguajes de programación POO, y una comprensión de al menos el 80% de la lógica de la programación orientada a objetos. Además de adquirir bases y conceptos de POO, se aprenderán técnicas de trabajo novedosas, manejo de IDEs y de herramientas dígitales, todo esto para facilitar la realización de aplicaciones de desarrollo de software.

Temas Abordados

HTML5

Durante el desarrollo de este tema tuvimos una actualización de las etiquetas de HTML. Aprendimos sobre la importancia de implementar las etiquetas HTML5 en nuestros sitios web, ya que estas etiquetas se basan en la semantica, y realizamos una página web utilizando estas etiquetas

POO

Repasamos los conceptos basicos de la POO para entender la lógica de la misma, ampliando la comprensión con algunos ejemplos, conocimos los principales lenguajes POO y el por qué se les considera orientados a objetos

JAVA

Aprendimos a utilizar los diferentes componentes y clases que nos proporciona este lenguaje de programación, para poder crear aplicaciones de software profesionales de forma estructurada y optimizada, al finalizar el curso estaremos preparados para desarollar herramientas utiles que utilicen la tecnología Java, para dar solución a distintos problemas.
